Given a string s, find the length of the longest substring without duplicate characters.

---

# Understand
題目提供一個s字串，要在字串中找到連續不重複字元的片段之最大長度。

所以要在字串中找一段子字串，且子字串中的字元皆不重複，然後回傳子字串的長度。

### Happy Case
|input|output|解釋|
|:----|:-----|:--|
|'aabcdd'|4|'abcd'|
|'pwdssdfkwv|6|'pwds'片段長度為4，'sdfkwv'片段長度6|

### Edge Case
|input|output|解釋|
|:----|:-----|:---|
|'a'|1|只有一個字元也算|
|''|0|空字串|
|'aaaa'|1|全部都相同字元，所以只回傳1|


# Match
題目要考的是 [滑動視窗](./12滑動視窗.md) + Hash Map(字元頻率統計)

透過寬度可動態收縮(造成滑動效果)，來檢查每次滑動後，視窗內的新片段字元是否重複出現在 Hash Map中，如果在Hash Map中則視窗寬度收縮，否則將新字元存入Hasp Map中並繼續向右伸展。

```
序列:  [ a  b  c  d  e  f  g ]
         ↑              ↑
        left           right
         └──────────────┘
              視窗 (window)

→ right 右移：視窗擴張，納入新元素
→ left  右移：視窗收縮，移除舊元素
```

# Plan
1. 使用一個指針left，作為視窗的左邊邊距。
2. 建立一個字典char_index，來儲存遍歷過的字元之出現的索引值
3. 建立一個變數max_length，儲存當前視窗的寬度，即是連續不重複的子字串長度
3. 遍歷s字串的長度，被選中的字元作為視窗的右邊邊距
4. 遍歷中的當前字元，
    - 若不存在於char_index，則加入，繼續下一個字元
    - 若存在於char_index且字元索引比left大，則更新left指向的字元，改成指向當前字元的下一個字元
    - 更新視窗寬度到max_length
5. 直到遍歷完s的所有字元，回傳max_length

# Implement
```python
def length_of_longest_substring(s: str) -> int:
    left = 0
    max_length = 0
    char_index = {} # 字元:index

    for right in range(len(s)):
        char = s[right]

        if char in char_index and char_index[char] >= left:
            left = char_index[char] + 1

        char_index[char] = right 

        max_length = max(max_length, right - left + 1)

    return max_length
```

# Review
1. Happy Case: s = 'aabcdd'
    ```
    初始化: left = 0, max_length = 0, char_index={}
    第一輪: right = 0
        char = s[0] = 'a'
        'a'不在char_index中，所以char_index = {'a':0}
        max_length = max(0, 0-0+1) = 1
    left = 0, mex_length = 1, char_index={'a':0}
    第二輪: right = 1
        char = s[1] = 'a'
        'a'存在於char_index中，所以left = char_index['a'] + 1 = 0 + 1 = 1
        更新char_index = {'a':1}
        max_length = max(1, 1-1+1) = 1
    left = 1, max_length = 1, char_index={'a':1}
    第三輪: right = 2
        char = s[2] = 'b'
        'b'不在char_index中，所以char_index = {'a':1, 'b':2}
        max_length = max(1, 2-1+1) = 2
    left = 1, max_length=2, char_index={'a':1, 'b':2} 
    第四輪: right = 3
        char = s[3] = 'c'
        'c'不在char_index中，所以char_index = {'a':1, 'b':2, 'c':3}
        max_length = max(1, 3-1+1) = 3
    以此類推
    ```
    最終輸出:4
2. Edge Case: s = 'aaaaa'
    ```
    初始化: left = 0, max_length = 0, char_index={}
    第一輪: right = 0
        char = s[0] = 'a'
        'a'不在char_index中，所以char_index = {'a':0}
        max_length = max(0, 0-0+1) = 1
    left = 0, mex_length = 1, char_index={'a':0}
    第二輪: right = 1
        char = s[1] = 'a'
        'a'存在於char_index中，所以left = char_index['a'] + 1 = 0 + 1 = 1
        更新char_index = {'a':1}
        max_length = max(1, 1-1+1) = 1
    left = 1, max_length = 1, char_index={'a':1}
    第三輪: right = 2
        char = s[2] = 'a'
        'a'存在於char_index中，所以left = char_index['a'] + 1 = 1 + 1 = 2
        更新char_index = {'a':2}
        max_length = max(1, 2-2+1) = 1
    left = 2, max_length = 1, char_index={'a':2}
    ```

# Evaluate
|複雜度|值|解釋|
|:----|:-|:---|
|時間複雜度|O(n)|right 指針遍歷字串一次，每個字元只處理一次|
|空間複雜度|O(min(n, m))|m 為字元集大小（如 ASCII=128），字典最多存 m 個字元|
