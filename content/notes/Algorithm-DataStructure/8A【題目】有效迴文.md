Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:
1. Open brackets must be closed by the same type of brackets.
2. Open brackets must be closed in the correct order.
3. Every close bracket has a corresponding open bracket of the same type.
----

# Understand
題目提供一個由`{`、`}`、`[`、`]`、`(`、`)`組成的字串。要判別這個字串是否同時滿足以下要求:
1. 左括弧的種類必須與右括弧種類一致
2. 左括號必須以正確的順序閉合
3. 每個右括號都有一個對應類型的左括號

### Happy Case
|input|output|解釋|
|:----|:-----|:---|
|'{}()[]'|true|每一種左括弧都對應相對的右括弧|
|'{()}'|true|左右括弧的順序能閉合|
|'{[}]'|false|兩種括弧無法順利閉合|

### Edge Case
|input|output|解釋|
|:----|:-----|:---|
|'['|false|括弧無法閉合|
|']'|false|括弧無法閉合|
|''|true|空字串|

# Match
題目要考的是 [堆疊](./8堆疊Stack.md) 的<mark>後進先出</mark>特性，而括號的特性正好也是【最近開啟的必須最先被關閉】。

# Plan
1. 若遇到左括弧，則送入堆疊中儲存
2. 若遇到右括弧，不會被送入堆疊中，而是拿最後一個送入堆疊的左括弧比對，如果兩者種類不一致，否則違反條件(return false)，所以同理遇到堆疊為空，也return False
3. 如果兩者種類一致，則被配對的左括弧從堆疊中移除
4. 檢查堆疊中是否還有殘留沒有被配對到的左括弧，有則違反規則，否則return true

# Implement
```python
def isValid(s: str) -> bool:
    stack = []

    for char in s:
        if char == '[' or char == '{' or char == '(':
            stack.append(char)
        elif char == ']' and stack and stack[-1] == '[':
            stack.pop()
        elif char == '}' and stack and stack[-1] == '{':
            stack.pop()
        elif char == ')' and stack and stack[-1] == '(':
            stack.pop()
        else:
            return False

    if stack:
        return False
    else:
        return True
```
